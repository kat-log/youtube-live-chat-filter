YouTube特別コメントフィルター 設定

YouTubeライブチャットで特別なコメントを抽出して表示する拡張機能の設定

API設定

YouTube Data API v3のキーが必要です。 Google Cloud Console で取得できます。

使い方

  1. 上記でYouTube Data API Keyを設定してください
  2. YouTubeのライブ配信ページを開いてください
  3. 拡張機能のアイコンをクリックしてポップアップを開いてください
  4. 「監視開始」ボタンをクリックして特別コメントの監視を開始してください
  5. モデレーター、メンバー（スポンサー）、配信者のコメントがリアルタイムで表示されます

対象コメントについて

配信者 チャンネルオーナー（配信者本人）のコメント
モデレーター 配信者が任命したモデレーターのコメント
メンバー チャンネルメンバー（スポンサー）のコメント

APIキー取得方法

  1. Google Cloud Consoleにアクセス
  2. 新しいプロジェクトを作成するか、既存のプロジェクトを選択
  3. 「APIとサービス」→「ライブラリ」から「YouTube Data API v3」を有効化
  4. 「APIとサービス」→「認証情報」から「APIキーを作成」
  5. 作成されたAPIキーをコピーして上記のフィールドに貼り付け

注意: APIキーは適切に管理し、他人と共有しないでください。 必要に応じてAPI使用量の制限を設定することをお勧めします。
